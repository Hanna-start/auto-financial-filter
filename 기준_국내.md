# 국내(코스피·코스닥) 선별 기준

> 실행 소스는 [`engine.py`]의 `KR_CRITERIA` 프로파일 + `screen()`. 이 문서는 그 기준의 설명본이며, 값을 바꾸려면 `engine.py`의 `KR_CRITERIA`를 수정한다. 미국 기준은 [기준_미국.md] 참고(이원화).

원래 '재무선배' 엄격 기준 그대로. 적용 대상: `run_kospi_quarterly.py`, `run_kosdaq_quarterly.py`.

## 3단계 게이트 (모두 AND)

| 단계 | 기준 | 값 |
|---|---|---|
| ① 유동성 | 최근 10거래일 평균 거래대금 | **≥ 100억원** |
| ② 재무건전 | 부채비율(총부채/총자본) | **< 200%** |
| | 매출성장 (YoY, 같은 분기 전년) | **≥ 10%** |
| | TTM 영업현금흐름 | **> 0** |
| | 자본총계 | **> 0** (음수자본 차단) |
| ③ 품질성장 | TTM 영업이익률 | **≥ 10%** |
| | 이익 피크 (최근 4분기 영업이익이 보유 이력 중 최고) | 충족 |
| | 원가율 개선 (최근 TTM 매출원가율 ≤ 가장 이른 4분기) | 충족 |

## 판단 원칙

- **규모·수익성 = TTM** (최근 4개 분기 합), **성장 = YoY** (같은 분기 전년 대비).
- 분기값은 **누적 차감**으로 순수 분기 환산 (Q2=반기−1Q, Q4=연간−3Q).
- 이력창 제한 없음(`recent_quarters=0`) — 국내 분기 데이터가 ~5년이라 자연히 최근 창.

## 표시용(필터 아님)

- PER·PBR·PSR·시총/TTM영업이익 + 현재가·등락률. **선별에는 미사용.**
- 분모(순이익·자본·매출)가 0에 가깝거나 결측이면 생기는 **비정상 배수(1,000배 초과)는 표시에서 제외('—')**. 정상 고배수는 보존. (`engine.RATIO_MAX`, KR·US 공통)

## 데이터

- 재무 = DART 분기(`dart-audit-extractor`의 `screener.db` `financials_q`).
- 시세 = KRX 일별(FinanceDataReader 캐시, 최근 거래일로 walk-back) + `market_cache` 폴백.
