# 🏦 재무 건전성 기반 주식 필터링 시스템 - 완전 프로젝트 정리

## 📋 프로젝트 개요

이 프로젝트는 AI를 활용하여 한국과 미국 주식 시장에서 재무적으로 건전한 기업을 자동으로 선별하는 시스템을 개발한 것입니다.

### 🎯 프로젝트 목표
- **한국 주식**: KOSPI/KOSDAQ 상장 기업 중 재무 건전성이 우수한 기업 선별
- **미국 주식**: NYSE/NASDAQ 상장 기업 중 재무 건전성이 우수한 기업 선별
- **3단계 필터링**: 유동성 → 재무건전성 → 품질성장 순서로 체계적 분석
- **엑셀 결과 출력**: 단계별 분석 결과를 Excel 파일로 자동 생성

## 📊 필터링 기준 상세 설명

### 🔄 1단계: 유동성 필터 (Liquidity Filter)
**목적**: 거래가 활발한 주식만 선별하여 투자 시 매매가 원활한 종목 확보

**한국 시장 기준**:
- 최소 거래대금: 500억원 (30일 평균)
- 최소 거래량: 일정 수준 이상

**미국 시장 기준**:
- 최소 거래대금: 5천만 달러 (30일 평균)
- 최소 거래량: 일정 수준 이상
### 💰 2단계: 재무건전성 필터 (Financial Health Filter)
**목적**: 부채가 적고 현금흐름이 안정적인 기업 선별

**공통 기준**:
- 부채비율: 200% 이하 (완화 시 400% 이하)
- 매출성장률: 5% 이상 (완화 시 -5% 이상)
- 영업현금흐름: 4분기 연속 양수

### 📈 3단계: 품질성장 필터 (Quality Growth Filter)
**목적**: 수익성이 개선되고 있는 고품질 기업 선별

**공통 기준**:
- 영업이익률: 5% 이상 (완화 시 0% 이상)
- 수익성 피크: 최근 4분기가 과거 4년 중 최고 수준
- 매출원가율 개선: 6분기 동안 전반적 감소 추세

### 🏗️ 1. 기본 데이터 모델 (models/base.py)

```python
"""
기본 데이터 모델 정의
- 주식 심볼, 재무 지표, 필터링 결과 등의 데이터 구조를 정의
"""

@dataclass
class StockSymbol:
    """주식 심볼 정보를 담는 클래스"""
    code: str        # 주식 코드 (예: "005930", "AAPL")
    name: str        # 회사명 (예: "삼성전자", "Apple Inc.")
    market: str      # 거래소 (예: "KOSPI", "NASDAQ")
    
    def is_korean_market(self) -> bool:
        """한국 시장 주식인지 확인"""
        return self.market in ['KOSPI', 'KOSDAQ']
    
    def is_us_market(self) -> bool:
        """미국 시장 주식인지 확인"""
        return self.market in ['NYSE', 'NASDAQ', 'AMEX', 'OTC']

@dataclass
class LiquidityMetrics:
    """유동성 지표 클래스"""
    symbol: StockSymbol          # 주식 심볼
    avg_daily_volume: float      # 일평균 거래량
    avg_daily_value: float       # 일평균 거래대금 (원화 기준)
    period_days: int             # 분석 기간 (일)

@dataclass
class FinancialMetrics:
    """재무건전성 지표 클래스"""
    symbol: StockSymbol                    # 주식 심볼
    debt_ratio: float                      # 부채비율 (%)
    operating_cash_flow: List[float]       # 영업현금흐름 (4분기)
    revenue_growth_yoy: float              # 전년동기 대비 매출성장률 (%)
    quarterly_revenue: List[float]         # 분기별 매출 (4분기)

@dataclass
class ProfitabilityMetrics:
    """수익성 및 성장성 지표 클래스"""
    symbol: StockSymbol                    # 주식 심볼
    operating_margin: float                # 영업이익률 (%)
    operating_profit_trend: List[float]    # 영업이익 추이 (16분기, 4년)
    cogs_ratio_trend: List[float]          # 매출원가율 추이 (6분기)
    is_profit_peak: bool                   # 최근 4분기가 수익 피크인지 여부

@dataclass
class FilterResult:
    """필터링 결과 클래스"""
    passed_symbols: List[StockSymbol]      # 통과한 주식 목록
    failed_symbols: List[StockSymbol]      # 탈락한 주식 목록
    stage: str                             # 필터링 단계명
    criteria_applied: Dict[str, Any]       # 적용된 기준
    
    @property
    def pass_rate(self) -> float:
        """통과율 계산 (백분율)"""
        if self.total_processed == 0:
            return 0.0
        return len(self.passed_symbols) / self.total_processed * 100
```

### 🏭 3. 메인 파이프라인 (pipeline.py)

```python
"""
주식 필터링 메인 파이프라인
- 3단계 필터를 순차적으로 실행하여 최종 후보 주식 선별
"""

class StockFilterPipeline:
    """주식 필터링 파이프라인 클래스"""
    
    def __init__(self, config: FilterConfig):
        """파이프라인 초기화"""
        self.config = config                    # 설정 정보
        self.filters: List[BaseFilter] = []     # 필터 목록
        self.logger = logging.getLogger(__name__)
    
    def add_filter(self, filter_instance: BaseFilter) -> None:
        """필터 추가"""
        self.filters.append(filter_instance)
        self.logger.info(f"Added filter: {filter_instance.get_stage_name()}")
    
    def execute(self, symbols: List[StockSymbol]) -> PipelineResult:
        """파이프라인 실행 - 전체 필터링 과정 수행"""
        start_time = time.time()
        
        # 초기 주식 목록
        current_symbols = symbols[:]
        stage_results = []
        
        self.logger.info(f"Starting pipeline with {len(symbols)} symbols")
        
        # 각 필터를 순차적으로 실행
        for i, filter_instance in enumerate(self.filters, 1):
            stage_name = filter_instance.get_stage_name()
            
            self.logger.info(f"Stage {i}: {stage_name} - Processing {len(current_symbols)} symbols")
            
            # 필터 실행
            filter_result = filter_instance.filter(current_symbols)
            stage_results.append(filter_result)
            
            # 다음 단계로 통과한 주식만 전달
            current_symbols = filter_result.passed_symbols
            
            # 단계별 결과 로깅
            self.logger.info(f"Stage {i} complete: {len(filter_result.passed_symbols)} passed, "
                           f"{len(filter_result.failed_symbols)} failed "
                           f"({filter_result.pass_rate:.1f}% pass rate)")
            
            # 통과한 주식이 없으면 조기 종료
            if not current_symbols:
                self.logger.warning(f"No symbols passed Stage {i}. Pipeline terminated early.")
                break
        
        # 실행 시간 계산
        execution_time = time.time() - start_time
        
        # 최종 결과 생성
        result = PipelineResult(
            initial_symbols=symbols,
            final_candidates=current_symbols,
            stage_results=stage_results,
            execution_time_seconds=execution_time
        )
        
        self.logger.info(f"Pipeline complete: {len(current_symbols)} final candidates "
                        f"from {len(symbols)} initial symbols ({execution_time:.2f}s)")
        
        return result
```
### 💧 4. 1단계 필터: 유동성 필터 (filters/liquidity_filter.py)

```python
"""
1단계 유동성 필터
- 거래량과 거래대금을 기준으로 유동성이 충분한 주식만 선별
- 투자 시 매매가 원활한 종목만 통과시킴
"""

class LiquidityFilter(BaseFilter):
    """유동성 필터 클래스"""
    
    def __init__(self, config: FilterConfig, data_manager):
        super().__init__(config)
        self.data_manager = data_manager        # 데이터 수집 관리자
        self.logger = logging.getLogger(__name__)
    
    def filter(self, symbols: List[StockSymbol]) -> FilterResult:
        """유동성 필터링 실행"""
        self.logger.info(f"Starting liquidity filtering for {len(symbols)} symbols")
        
        passed_symbols = []     # 통과한 주식 목록
        failed_symbols = []     # 탈락한 주식 목록
        processed_count = 0     # 처리된 주식 수
        
        # 적용된 기준 정보
        criteria_applied = {
            "min_trading_volume_krw": self.config.min_trading_volume_krw,
            "trading_volume_period_days": self.config.trading_volume_period_days
        }
        
        # 각 주식에 대해 유동성 검사
        for symbol in symbols:
            try:
                processed_count += 1
                
                # 진행률 표시 (10개씩)
                if processed_count % 10 == 0 or processed_count == len(symbols):
                    progress_pct = (processed_count / len(symbols)) * 100
                    self.logger.info(f"Progress: {processed_count}/{len(symbols)} ({progress_pct:.1f}%)")
                
                # 유동성 지표 계산
                liquidity_metrics = self._get_liquidity_metrics(symbol)
                
                # 유동성 기준 검사
                if self._meets_liquidity_criteria(liquidity_metrics):
                    passed_symbols.append(symbol)
                    self.logger.debug(f"PASS: {symbol.code} - Volume: {liquidity_metrics.avg_daily_value:,.0f}")
                else:
                    failed_symbols.append(symbol)
                    self.logger.debug(f"FAIL: {symbol.code} - Volume: {liquidity_metrics.avg_daily_value:,.0f}")
                    
            except Exception as e:
                # 오류 발생 시 해당 주식은 탈락 처리
                failed_symbols.append(symbol)
                self.logger.warning(f"Error processing {symbol.code}: {str(e)}")
                continue
        
        # 결과 생성
        result = FilterResult(
            passed_symbols=passed_symbols,
            failed_symbols=failed_symbols,
            stage="filtered_symbols_step1",  # 1단계 식별자
            criteria_applied=criteria_applied
        )
        
        self.logger.info(f"Liquidity filtering complete: {len(passed_symbols)} passed, "
                        f"{len(failed_symbols)} failed ({result.pass_rate:.1f}% pass rate)")
        
        return result
    
    def _get_liquidity_metrics(self, symbol: StockSymbol) -> LiquidityMetrics:
        """주식의 유동성 지표 계산"""
        # 거래 데이터 수집 (30일)
        trading_data = self.data_manager.get_trading_data(
            symbol, 
            self.config.trading_volume_period_days
        )
        
        if trading_data.empty:
            raise ValueError(f"No trading data available for {symbol.code}")
        
        # 일평균 거래량 및 거래대금 계산
        avg_daily_volume = trading_data['Volume'].mean()
        avg_daily_value = trading_data['TradingValue'].mean()
        
        return LiquidityMetrics(
            symbol=symbol,
            avg_daily_volume=avg_daily_volume,
            avg_daily_value=avg_daily_value,
            period_days=len(trading_data)
        )
    
    def _meets_liquidity_criteria(self, metrics: LiquidityMetrics) -> bool:
        """유동성 기준 충족 여부 검사"""
        # 최소 거래대금 기준 검사
        volume_ok = metrics.avg_daily_value >= self.config.min_trading_volume_krw
        
        self.logger.debug(f"Liquidity check for {metrics.symbol.code}: "
                         f"Volume={volume_ok} ({metrics.avg_daily_value:,.0f} >= {self.config.min_trading_volume_krw:,.0f})")
        
        return volume_ok
```
### 🏦 5. 2단계 필터: 재무건전성 필터 (filters/financial_health_filter.py)

```python
"""
2단계 재무건전성 필터
- 부채비율, 매출성장률, 현금흐름을 기준으로 재무적으로 건전한 기업 선별
- 안정적인 재무구조를 가진 기업만 통과시킴
"""

class FinancialHealthFilter(BaseFilter):
    """재무건전성 필터 클래스"""
    
    def __init__(self, config: FilterConfig, data_manager):
        super().__init__(config)
        self.data_manager = data_manager
        self.logger = logging.getLogger(__name__)
        
        # 초기화 시 설정 정보 로깅
        self.logger.info("FinancialHealthFilter initialized with:")
        self.logger.info(f"  Max debt ratio: {config.max_debt_ratio_percent}%")
        self.logger.info(f"  Min revenue growth: {config.min_revenue_growth_percent}%")
        self.logger.info(f"  Cash flow quarters: {config.cash_flow_quarters}")
    
    def filter(self, symbols: List[StockSymbol]) -> FilterResult:
        """재무건전성 필터링 실행"""
        self.logger.info(f"Starting financial health filtering for {len(symbols)} symbols")
        
        passed_symbols = []
        failed_symbols = []
        
        # 적용된 기준 정보
        criteria_applied = {
            "max_debt_ratio_percent": self.config.max_debt_ratio_percent,
            "min_revenue_growth_percent": self.config.min_revenue_growth_percent,
            "cash_flow_quarters_required": self.config.cash_flow_quarters
        }
        
        # 각 주식에 대해 재무건전성 검사
        for i, symbol in enumerate(symbols, 1):
            try:
                self.logger.info(f"Processing symbol {i}/{len(symbols)}: {symbol.code}")
                
                # 재무 지표 계산
                financial_metrics = self._get_financial_metrics(symbol)
                
                # 재무건전성 기준 검사
                if self._meets_financial_health_criteria(financial_metrics):
                    passed_symbols.append(symbol)
                    self.logger.debug(f"PASS: {symbol.code} - Debt: {financial_metrics.debt_ratio:.1f}%, "
                                    f"Growth: {financial_metrics.revenue_growth_yoy:.1f}%")
                else:
                    failed_symbols.append(symbol)
                    self.logger.debug(f"FAIL: {symbol.code} - Debt: {financial_metrics.debt_ratio:.1f}%, "
                                    f"Growth: {financial_metrics.revenue_growth_yoy:.1f}%")
                    
            except Exception as e:
                # 오류 발생 시 해당 주식은 탈락 처리
                failed_symbols.append(symbol)
                self.logger.warning(f"Error processing {symbol.code}: {str(e)}")
                continue
        
        # 결과 생성
        result = FilterResult(
            passed_symbols=passed_symbols,
            failed_symbols=failed_symbols,
            stage="Financial Health Filter",  # 2단계 식별자
            criteria_applied=criteria_applied
        )
        
        self.logger.info(f"Financial health filtering complete: {len(passed_symbols)} passed, "
                        f"{len(failed_symbols)} failed")
        
        return result
    
    def _get_financial_metrics(self, symbol: StockSymbol) -> FinancialMetrics:
        """주식의 재무건전성 지표 계산"""
        # 재무제표 데이터 수집 (4분기)
        financial_data = self.data_manager.get_financial_data(symbol)
        
        # 데이터 형식에 따른 처리 (한국: DataFrame, 미국: Dictionary)
        if hasattr(financial_data, 'empty'):
            # DataFrame 형식 (한국 시장)
            if financial_data.empty:
                raise ValueError(f"No financial data available for {symbol.code}")
            
            # 최신 분기 데이터에서 부채비율 계산
            latest_data = financial_data.iloc[-1]
            total_debt = latest_data.get('TotalDebt', 0)
            total_equity = latest_data.get('TotalEquity', 1)
            debt_ratio = (total_debt / total_equity) * 100 if total_equity > 0 else 999.0
            
            # 분기별 매출 및 현금흐름 추출
            quarterly_revenue = financial_data['Revenue'].tolist()[-4:]
            operating_cash_flow = financial_data['OperatingCashFlow'].tolist()[-4:]
            
        else:
            # Dictionary 형식 (미국 시장)
            quarterly_data = financial_data.get('quarterly_data', [])
            if not quarterly_data:
                raise ValueError(f"No financial data available for {symbol.code}")
            
            # 최신 분기 부채비율
            latest_quarter = quarterly_data[0]
            debt_ratio = latest_quarter.get('debt_ratio', 0)
            
            # 분기별 데이터 추출
            quarterly_revenue = [q.get('revenue', 0) for q in quarterly_data]
            operating_cash_flow = [q.get('operating_cash_flow', 0) for q in quarterly_data]
        
        # 매출성장률 계산 (전년동기 대비)
        if len(quarterly_revenue) >= 4:
            current_revenue = quarterly_revenue[-1]  # 최근 분기
            previous_year_revenue = quarterly_revenue[0]  # 1년 전 분기
            
            if previous_year_revenue > 0:
                revenue_growth_yoy = ((current_revenue - previous_year_revenue) / previous_year_revenue) * 100
            else:
                revenue_growth_yoy = 0.0
        else:
            revenue_growth_yoy = 0.0
        
        return FinancialMetrics(
            symbol=symbol,
            debt_ratio=debt_ratio,
            operating_cash_flow=operating_cash_flow,
            revenue_growth_yoy=revenue_growth_yoy,
            quarterly_revenue=quarterly_revenue
        )
    
    def _meets_financial_health_criteria(self, metrics: FinancialMetrics) -> bool:
        """재무건전성 기준 충족 여부 검사"""
        # 기준 1: 부채비율 검사
        debt_ok = metrics.debt_ratio <= self.config.max_debt_ratio_percent
        
        # 기준 2: 매출성장률 검사
        growth_ok = metrics.revenue_growth_yoy >= self.config.min_revenue_growth_percent
        
        # 기준 3: 영업현금흐름 검사 (4분기 모두 양수)
        cash_flow_ok = all(cf > 0 for cf in metrics.operating_cash_flow)
        
        self.logger.debug(f"Financial health check for {metrics.symbol.code}: "
                         f"Debt={debt_ok} ({metrics.debt_ratio:.1f}%), "
                         f"Growth={growth_ok} ({metrics.revenue_growth_yoy:.1f}%), "
                         f"CashFlow={cash_flow_ok}")
        
        return debt_ok and growth_ok and cash_flow_ok
```

### 📈 6. 3단계 필터: 품질성장 필터 (filters/quality_growth_filter.py)

```python
"""
3단계 품질성장 필터
- 영업이익률, 수익성 트렌드, 매출원가율 개선을 기준으로 고품질 성장 기업 선별
- 지속적으로 수익성이 개선되고 있는 기업만 통과시킴
"""

class QualityGrowthFilter(BaseFilter):
    """품질성장 필터 클래스"""
    
    def __init__(self, config: FilterConfig, data_manager):
        super().__init__(config)
        self.data_manager = data_manager
        self.logger = logging.getLogger(__name__)
    
    def filter(self, symbols: List[StockSymbol]) -> FilterResult:
        """품질성장 필터링 실행"""
        self.logger.info(f"Starting quality growth filtering for {len(symbols)} symbols")
        
        passed_symbols = []
        failed_symbols = []
        processed_count = 0
        
        # 적용된 기준 정보
        criteria_applied = {
            "min_operating_margin_percent": self.config.min_operating_margin_percent,
            "profit_trend_years": self.config.profit_trend_years,
            "cogs_trend_quarters": self.config.cogs_trend_quarters
        }
        
        # 각 주식에 대해 품질성장 검사
        for symbol in symbols:
            try:
                processed_count += 1
                
                # 진행률 표시 (5개씩)
                if processed_count % 5 == 0 or processed_count == len(symbols):
                    progress_pct = (processed_count / len(symbols)) * 100
                    self.logger.info(f"Progress: {processed_count}/{len(symbols)} ({progress_pct:.1f}%)")
                
                # 수익성 지표 계산
                profitability_metrics = self._get_profitability_metrics(symbol)
                
                # 품질성장 기준 검사
                if self._meets_quality_growth_criteria(profitability_metrics):
                    passed_symbols.append(symbol)
                    self.logger.debug(f"PASS: {symbol.code} - OPM: {profitability_metrics.operating_margin:.1f}%, "
                                    f"Peak: {profitability_metrics.is_profit_peak}")
                else:
                    failed_symbols.append(symbol)
                    self.logger.debug(f"FAIL: {symbol.code} - OPM: {profitability_metrics.operating_margin:.1f}%, "
                                    f"Peak: {profitability_metrics.is_profit_peak}")
                    
            except Exception as e:
                # 오류 발생 시 해당 주식은 탈락 처리
                failed_symbols.append(symbol)
                self.logger.warning(f"Error processing {symbol.code}: {str(e)}")
                continue
        
        # 결과 생성
        result = FilterResult(
            passed_symbols=passed_symbols,
            failed_symbols=failed_symbols,
            stage="final_candidate_list",  # 3단계 식별자
            criteria_applied=criteria_applied
        )
        
        self.logger.info(f"Quality growth filtering complete: {len(passed_symbols)} passed, "
                        f"{len(failed_symbols)} failed ({result.pass_rate:.1f}% pass rate)")
        
        return result
    
    def _analyze_cogs_trend(self, cogs_ratios: List[float]) -> bool:
        """매출원가율 개선 트렌드 분석 - 핵심 개선 로직"""
        if len(cogs_ratios) != 6:
            return False
        
        # 현실적인 트렌드 분석: 전반부 vs 후반부 평균 비교
        # 분기별 변동을 허용하면서 전체적인 개선 추세 확인
        first_half_avg = sum(cogs_ratios[:3]) / 3   # 초기 3분기 평균
        second_half_avg = sum(cogs_ratios[3:]) / 3  # 최근 3분기 평균
        
        # 최신 분기와 가장 오래된 분기 비교
        recent_vs_oldest = cogs_ratios[-1] < cogs_ratios[0]
        
        # 다음 중 하나라도 만족하면 개선으로 판단:
        # 1. 후반부 평균이 전반부보다 낮음 (전체적 개선)
        # 2. 최신 분기가 가장 오래된 분기보다 낮음 (장기적 개선)
        overall_improvement = second_half_avg < first_half_avg
        long_term_improvement = recent_vs_oldest
        
        return overall_improvement or long_term_improvement
    
    def _meets_quality_growth_criteria(self, metrics: ProfitabilityMetrics) -> bool:
        """품질성장 기준 충족 여부 검사"""
        # 기준 1: 영업이익률 >= 설정값
        margin_ok = metrics.operating_margin >= self.config.min_operating_margin_percent
        
        # 기준 2: 최근 4분기가 수익 피크
        peak_ok = metrics.is_profit_peak
        
        # 기준 3: 매출원가율 개선 트렌드
        cogs_trend_ok = self._analyze_cogs_trend(metrics.cogs_ratio_trend)
        
        self.logger.debug(f"Quality criteria for {metrics.symbol.code}: "
                         f"Margin={margin_ok} ({metrics.operating_margin:.1f}%), "
                         f"Peak={peak_ok}, COGS_trend={cogs_trend_ok}")
        
        return margin_ok and peak_ok and cogs_trend_ok
```

```python
class SECFinancialAdapter(DataSourceAdapter):

                # 매출원가

                    'total_equity': total_equity,
                    'operating_cash_flow': operating_cash_flow,
                    'cogs': cogs,
                    'debt_ratio': debt_ratio
                })
```

### 📊 8. Excel 출력 기능 (utils/export.py)

```python
"""
Excel 파일 출력 기능
- 필터링 결과를 Excel 파일로 내보내기
- 단계별 분석 결과를 시트별로 구성

class DataExporter:

            import pandas as pd

            with pd.ExcelWriter(filename, engine='openpyxl') as writer:

                stage_summary_data = []
                for i, stage_result in enumerate(result.stage_results, 1):

                                '종목명': symbol.name,

                            })
```

### 2. 📈 COGS 트렌드 분석 로직 개선

**문제**: 기존 로직은 6분기 모두 연속 감소를 요구하여 너무 엄격함 (실제 기업은 분기별 변동 존재)

**해결책**: 현실적인 트렌드 분석으로 변경

```python
def _analyze_cogs_trend(self, cogs_ratios: List[float]) -> bool:
    """개선된 COGS 트렌드 분석"""
    if len(cogs_ratios) != 6:
        return False
    
    # 전반부 vs 후반부 평균 비교 (분기별 변동 허용)
    first_half_avg = sum(cogs_ratios[:3]) / 3   # 초기 3분기 평균
    second_half_avg = sum(cogs_ratios[3:]) / 3  # 최근 3분기 평균
    
    # 최신 분기와 가장 오래된 분기 비교
    recent_vs_oldest = cogs_ratios[-1] < cogs_ratios[0]
    
    # 다음 중 하나라도 만족하면 개선으로 판단:
    # 1. 후반부 평균이 전반부보다 낮음 (전체적 개선)
    # 2. 최신 분기가 가장 오래된 분기보다 낮음 (장기적 개선)
    overall_improvement = second_half_avg < first_half_avg
    long_term_improvement = recent_vs_oldest
    
    return overall_improvement or long_term_improvement
```

특히 3단계 필터링 시스템을 통해 체계적이고 과학적인 종목 선별이 가능하며, 실제 데이터를 활용한 분석으로 투자 의사결정에 도움이 되는 객관적인 정보를 제공합니다.

이 시스템은 앞으로도 지속적으로 개선하고 확장할 수 있는 기반을 마련했으며, 다른 시장이나 새로운 분석 기법을 추가하여 더욱 강력한 투자 도구로 발전시킬 수 있습니다.

**🎯 핵심 성과 요약**:
- ✅ 완전한 한국/미국 주식 필터링 시스템 구축
- ✅ 실시간 데이터 연동 및 Excel 자동 출력
- ✅ 109개 테스트 케이스로 검증된 안정성
- ✅ 확장 가능한 모듈형 아키텍처
- ✅ 실제 투자에 활용 가능한 실용성

이 프로젝트를 통해 AI를 활용한 체계적인 소프트웨어 개발 방법론을 익혔으며, 복잡한 금융 데이터를 다루는 실무 경험을 쌓을 수 있었습니다.
