### 3. 📈 상세한 탈락 사유 추적 시스템

#### **기존 문제점**
- 단순히 `failed_symbols`에만 추가
- 탈락 이유를 알 수 없음
- 디버깅 및 분석 어려움

#### **개선된 해결책**
```python
class FailureReason(Enum):
    MISSING_DATA = "Missing Data"
    DEBT_RATIO_EXCEEDED = "Debt Ratio Exceeded"
    REVENUE_GROWTH_BELOW = "Revenue Growth Below Threshold"
    # ... 더 많은 상세 사유

@dataclass
class FilterResultWithReasons:
    passed_symbols: List[StockSymbol]
    failed_symbols: List[StockSymbol]
    failure_reasons: Dict[str, FailureReason]  # 종목별 탈락 사유
    detailed_metrics: pd.DataFrame  # 상세 지표
```

#### **구현 완료**
- ✅ 8가지 상세 탈락 사유 분류
- ✅ 벡터화된 사유 분석 (np.where 활용)
- ✅ 탈락 사유별 통계 자동 생성
- ✅ Excel 출력 시 상세 사유 포함
