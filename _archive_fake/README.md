# _archive_fake — 가짜 데이터 기반 산출물 격리 보관소

이 폴더의 파일들은 **가짜(합성) 재무 데이터로 만들어진 구(舊) 산출물**입니다.
신뢰할 수 없으므로 작업 루트에서 격리했습니다. 언제든 삭제해도 무방하며,
필요하면 다시 꺼내 쓸 수 있도록 지우지 않고 보관만 했습니다.

| 파일 | 정체 |
|---|---|
| `재무건전성_필터링_결과_*.xlsx` (2) | 가짜 데이터 기반 구 시연 결과 — 신뢰 불가 |
| `Vectorized_US_Stock_Analysis_*.xlsx` | 합성 미국 데이터 기반 결과 — 신뢰 불가 |
| `build_modern_ui.py` | 위 가짜 xlsx를 읽어 대시보드를 만들던 스크립트 |
| `modern_dashboard.html`, `stock_dashboard.html` | 가짜 데이터로 렌더된 대시보드 |
| `check_samsung.py` | 가짜 xlsx를 들여다보던 스크래치 |
| `refactor.py` | 옛 폴더 경로(`...-v2_gemini advised`)를 가리키던 일회용 스크립트 |

> 진짜 산출물은 루트의 `상장사_스크리너_결과_*.xlsx`(실제 DART 재무, 현재 약 65개사 한정)입니다.
